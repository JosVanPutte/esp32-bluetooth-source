<B>Release 0.1:</B><BR>
A simple project that uses a webpage to configure the output to a bluetooth device.<BR>
A standard ESP32 is used without additional hardware<BR>
<BR>
Used the following libraries and plugins:<BR>
WiFiManager:<BR>
https://www.youtube.com/watch?v=VnfX9YJbaU8<BR>
LittleFs uploader:<BR>
https://randomnerdtutorials.com/arduino-ide-2-install-esp32-littlefs/<BR>
ESP32 as a Bluetooth source:<BR>
https://www.pschatzmann.ch/home/2020/09/15/sending-sound-from-an-esp32-to-a-bluetooth-sink-e-g-bluetooth-speaker/<BR>
look at this project:<BR>https://github.com/schreibfaul1/ESP32_A2DP_Source<BR>
<BR>
